## Description

Format document content in specified format (JSON | String | XML).

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```
Once the application is launched, please visit `http://localhost:3000/api`

## Run tests

```bash
# unit tests
$ npm run test
```
